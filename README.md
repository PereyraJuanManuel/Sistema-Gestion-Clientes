# Sistema de Gestión de Clientes

Una aplicación de escritorio en Java para registro y control de clientes, implementando operaciones CRUD (Crear, Leer, Actualizar, Eliminar). Parte de mi proyecto académico en la UTN.

## Características
- Gestión básica de clientes con ID, nombre y email.
- Operaciones CRUD en consola.
- Preparado para integración con MySQL (futuras versiones).

## Tecnologías
- Java
- (Próximamente: MySQL para persistencia)

## Instalación y Uso
1. Clona el repositorio: `git clone https://github.com/tu-usuario/cliente-management-system.git`
2. Abre en NetBeans o Eclipse.
3. Ejecuta `Main.java`.
4. Sigue las opciones en consola.
